# AI Content Marketing using UGC Ads

## Overview
This repository contains prompt engineering work for generating high-quality User-Generated Content (UGC) advertisements using AI. The prompts are designed to create authentic, engaging, and conversion-focused marketing content for businesses.

## Objective
- Generate professional UGC ad scripts.
- Create engaging social media marketing content.
- Demonstrate prompt engineering techniques.
- Produce marketing outputs suitable for Instagram Reels, TikTok, and YouTube Shorts.

## Repository Structure
```
README.md
prompts.md
outputs.md
screenshots/
```

## Tools Used
- ChatGPT
- GitHub
- Markdown

## Files
- **prompts.md** – Contains the prompts used for generating UGC advertisements.
- **outputs.md** – Contains the AI-generated marketing content.
- **screenshots/** – Screenshots demonstrating the prompt engineering process.
